# LRC Circuit Lab

General Physics II — Lab Experiment #9
Determines the resonance frequency of an LRC circuit and validates AC circuit theory via nonlinear least-squares curve fitting.

## Overview

The experiment measures peak current vs. frequency for an LRC series circuit, both with and without an iron core in the inductor. Logger Pro exports raw time-domain + FFT data as CSV files. Two Python scripts process those files and fit the LRC current model:

```
I(f) = alpha / sqrt( beta^2 * (f - f0^2/f)^2 + 1 ) + io
```

- `f0` — resonance frequency (Hz)
- `alpha` — peak current (A)
- `beta` — width parameter (proportional to L/R)
- `io` — current offset (A)

## Setup

Requires Python ≥ 3.10 and [uv](https://docs.astral.sh/uv/).

```bash
# Clone the repo
git clone <repo-url>
cd "LRC Lab"

# Create virtual environment and install dependencies
uv sync
```

`uv sync` reads `pyproject.toml` and `uv.lock`, creates `.venv/`, and installs `numpy`, `scipy`, and `matplotlib` at the locked versions.

## Usage

### Step 1 — process raw data

```bash
uv run python process_lrc.py
```

Reads every `{freq}nc.csv` / `{freq}wc.csv` from `lab_data/`, extracts the peak current amplitude and measured frequency (via FFT peak) for each signal-generator frequency, and writes `output.csv`.

### Step 2 — fit and plot

```bash
uv run python lrc_fit.py
```

Reads `output.csv`, fits the LRC model to both the no-core and with-core datasets using Nelder-Mead minimization, prints the fit parameters to the terminal, and saves the two-panel plot to `lrc_fit.png`.

#### Fit parameters printed to terminal

```
========================================
  Fit results — No Core
========================================
  f0    = 87.34 Hz   (resonance frequency)
  alpha = 0.267800 A   (peak current)
  beta  = 0.005123     (width / L·R parameter)
  io    = 0.000412 A   (current offset)
```

## Lab data file naming convention

Files in `lab_data/` follow the pattern `{sig_gen_freq}{condition}.csv`:

| Suffix | Meaning |
|--------|---------|
| `nc`   | No iron core in inductor |
| `wc`   | With iron core in inductor |

Example: `75wc.csv` → 75 Hz signal generator setting, iron core inserted.

Files are Logger Pro exports containing both time-domain current samples and FFT columns. The script extracts:
- **Measured frequency** — frequency bin with maximum FFT magnitude
- **Peak current** — maximum absolute value of the time-domain current signal

## Theoretical resonance

The resonance frequency predicted from component values is:

```
f_res = 1 / (2π √(LC))
```

Compare the curve-fit `f0` values against this formula using your measured L and C from the LRC meter.
